
# CyberShield AI

CyberShield AI is a real-time scam detection platform that identifies fraudulent messages
such as UPI phishing, bank KYC scams, courier fraud, and fake traffic challans using
machine learning and deep learning models.

## Features
- Multi-class scam detection
- NLP-based text analysis
- High-accuracy ML and BERT models
- Real-time inference via FastAPI
- Explainable predictions with confidence scores

## Tech Stack
Python, Scikit-learn, BERT, FastAPI, PyTorch
